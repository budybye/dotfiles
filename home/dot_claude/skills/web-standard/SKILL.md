# Web Standards Development Skill

This skill provides guidance on web standards including HTML, CSS, JavaScript, accessibility, performance, and security best practices for modern web development.

## Capabilities

- HTML5 semantic markup and validation
- CSS3 layout techniques (Flexbox, Grid, Custom Properties)
- Modern JavaScript (ES6+) and DOM manipulation
- Web Accessibility (WCAG, ARIA, inclusive design)
- Web Performance optimization (loading, rendering, runtime)
- Web Security best practices (CSP, CORS, sanitization)
- Responsive and mobile-first design principles
- Progressive Web App (PWA) fundamentals
- Cross-browser compatibility strategies

## Usage

Activate this skill when working on:
- Frontend web development projects
- Accessibility audits or implementations
- Performance optimization tasks
- Cross-browser compatibility issues
- Web standards compliance reviews
- Modern HTML/CSS/JavaScript features
- Responsive design implementations

## Best Practices

1. Follow W3C standards and WCAG guidelines
2. Prioritize semantic HTML for accessibility
3. Use progressive enhancement techniques
4. Implement responsive design with mobile-first approach
5. Optimize performance with efficient CSS and JavaScript
6. Ensure cross-browser compatibility
7. Maintain clean, well-documented code
8. Test with assistive technologies
