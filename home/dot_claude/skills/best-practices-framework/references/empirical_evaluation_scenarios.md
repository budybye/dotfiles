# Empirical Evaluation of Best Practices Skill

## Scenario 1: Frontend Framework Selection for New Project

**Context**: A startup is building a new web application and needs to select a frontend framework. The team consists of junior to mid-level developers with some React experience.

**Requirements**:
[critical] Easy learning curve for junior developers
[critical] Strong community support and documentation
[critical] Good performance for a complex SPA
Good ecosystem of third-party libraries
Active maintenance and regular updates

**Task**: Using the best-practis skill, evaluate and recommend a frontend framework for this project.

## Scenario 2: Backend Architecture for Scalable API

**Context**: A company needs to redesign their backend to handle increasing traffic. They currently have a monolithic PHP application that's becoming difficult to maintain.

**Requirements**:
[critical] Ability to handle 10x current traffic
[critical] Modular architecture for easier maintenance
[critical] Good developer onboarding experience
Support for microservices pattern
Integration with existing database systems

**Task**: Using the best-practis skill, propose a backend architecture and technology stack for this scenario.

## Scenario 3: Development Toolchain Modernization

**Context**: An enterprise development team wants to modernize their development workflow with better tools for CI/CD, testing, and collaboration.

**Requirements**:
[critical] Integration with existing Git workflow
[critical] Automated testing capabilities
[critical] Deployment automation with rollback support
Improved developer collaboration features
Monitoring and observability tools

**Task**: Using the best-practis skill, recommend a modern development toolchain for this team.
