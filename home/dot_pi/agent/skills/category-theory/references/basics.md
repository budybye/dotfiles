## 圏 (Category)

- **オブジェクト**: 集合、型、位相空間など任意の「もの」
- **射 (morphism)**: オブジェクト間の構造保存写像
- **合成**: `g ∘ f` が常に定義され、結合律 `h ∘ (g ∘ f) = (h ∘ g) ∘ f` が成り立つ
- **単位射**: 各オブジェクト `X` に対し `id_X : X → X` が存在し、`f ∘ id_X = f`、`id_Y ∘ f = f`

> **例**: **集合の圏** `Set`
> - オブジェクト: 任意の集合
> - 射: 関数
> - 合成: 関数合成
> - 単位射: 恒等関数

## 函手 (Functor)

- 対象写像: 各オブジェクト `X` を `F(X)` に写す
- 射写像: 各射 `f: X → Y` を `F(f): F(X) → F(Y)` に写す
- 合成・単位射を保つ: `F(g ∘ f) = F(g) ∘ F(f)`、`F(id_X) = id_{F(X)}`

> **例**: **忘却函手** `U: Grp → Set` は群を集合に忘却する。

## 自然変換 (Natural Transformation)

- 2つの函手 `F, G : C → D` の間の射の族 `η_X : F(X) → G(X)` が対象ごとに存在し、任意の射 `f: X → Y` に対し
  `G(f) ∘ η_X = η_Y ∘ F(f)` が成り立つ。

> **例**: 群の包含 `η_X: X → U(F(X))`（自由群から忘却された集合への射）

## 極限 (Limits) と余極限 (Colimits)

- **極限**は対象の**普遍的な受け取り**で、典型例は積 (product) と終対象 (terminal object)。
- **余極限**は対象の**普遍的な送り出し**で、典型例は余積 (coproduct) と始対象 (initial object)。
