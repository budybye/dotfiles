## 演習問題一覧

### P0 – 基礎

**問題 1**: 圏 `Set` のオブジェクトと射を列挙し、単位射と合成の性質を説明せよ。

**問題 2**: 函手 `U : Grp → Set`（群の忘却）について、対象写像と射写像を具体的に示し、`U(id_G) = id_{U(G)}` が成り立つことを確認せよ。

### P1 – 中級

**問題 3**: 圏 `C` の対象 `A` と `B` の積 `A × B` が存在すると仮定する。任意の対象 `X` と射 `f : X → A`、`g : X → B` に対し、唯一の射 `⟨f,g⟩ : X → A × B` が存在することを証明せよ。

**問題 4**: 函手 `F : C → D` と `G : D → C` が随伴 `F ⊣ G` であるとき、単位 `η : id_C ⇒ G∘F` と余単位 `ε : F∘G ⇒ id_D` の自然性方程式を書き下し、簡単な例（集合の自由モナド）で示せ。

### P2 – 発展

**問題 5**: 圏 `C` における終対象 `1` の普遍性を用いて、任意の射 `f : X → 1` が唯一であることを示す。

**問題 6**: Yoneda 補題を用いて、任意の函手 `F : C → Set` と対象 `A` に対し、自然変換 `η : Hom_C(A,-) ⇒ F` がどのように `F(A)` の元に対応するか具体例で説明せよ（例: `C = Set`、`A = {0,1}`）。

### P3 – 挑戦

**問題 7**: 圏 `C` が小限（finite limits）を持つと仮定する。任意の双射（equalizer）`e : E → X` が存在することを示す手順を概説せよ。

**問題 8**: 任意の圏 `C` と函手 `F : C → D` があるとき、`F` が全射（essentially surjective）であることと、Yoneda 埋め込み `y : C → [C^op, Set]` が保たれることの関係を議論せよ。
