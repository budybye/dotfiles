## Yoneda 補題 (Yoneda Lemma)

- **声明**: 任意の圏 `C` と対象 `A ∈ C`、そして `Set` への函手 `F : C → Set` に対し、自然変換の集合 `Nat(Hom_C(A, -), F)` と `F(A)` は自然同型である。
- **記号**: `Nat(Hom_C(A, -), F) ≅ F(A)`
- **概要**: 任意の自然変換 `η : Hom_C(A, -) ⇒ F` は `η_A(id_A)` によって一意に決定され、逆写像は `x ∈ F(A)` を `η_X(f) = F(f)(x)` と定義して得られる。

## 随伴 (Adjunction)

- **声明**: 2つの函手 `F : C → D`、`G : D → C` が随伴であるとは、自然同型 `Hom_D(F(X), Y) ≅ Hom_C(X, G(Y))` がすべての `X ∈ C`、`Y ∈ D` に対して成立することである。
- **左随伴**: `F ⊣ G` のとき `F` を左随伴、`G` を右随伴という。
- **単位・余単位**: 随伴に伴う自然変換 `η : id_C ⇒ G∘F`（単位）と `ε : F∘G ⇒ id_D`（余単位）

## 極限と余極限の基本定理

- **積 (Product)**: 二対象 `A, B` の積 `A × B` は普遍的な射 `π₁ : A × B → A`、`π₂ : A × B → B` を持ち、任意の `X` から `A`、`B` への射 `f : X → A`、`g : X → B` が唯一の射 `⟨f,g⟩ : X → A × B` に分解できる。
- **余積 (Coproduct)**: 二対象 `A, B` の余積 `A ⊔ B` は射 `ι₁ : A → A ⊔ B`、`ι₂ : B → A ⊔ B` を持ち、任意の `X` への射 `f : A → X`、`g : B → X` が唯一の射 `[f,g] : A ⊔ B → X` に合わせられる。
- **終対象 (Terminal Object)**: 任意の `X` から唯一の射がある対象 `1`。
- **始対象 (Initial Object)**: 任意の `X` への唯一の射がある対象 `0`。
